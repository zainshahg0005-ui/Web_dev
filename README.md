## Overview
This project demonstrates the use of HTML and CSS by implementing three different styling approaches (Style A, Style B, and Style C) .

---

## Files Included

- `index.html` → Main HTML file containing 6 box elements
- `StyleA.css` → Vertical layout with equal spacing
- `StyleB.css` → Horizontal layout with fixed positioning
- `styleC.css` → Extra credit stylesheet using CSS3 features
